# Project Summary

Goal: Vytvoř jednu samostatnou českou HTML stránku o Praze s výrazným hero blokem, moderním layoutem sekcí, kartami pro 3 obsahové bloky (památky, atmosféra a kultura, proč Prahu navštívit), krátkým úvodem a stručným shrnutím. Nepoužívej placeholder text. Použij lokální obrázky z assets/images a vlož je do výsledného HTML s alt texty.
Task type: landing-page

## Co bylo vytvořeno
- Statický web připravený pro publish přes GitHub Pages.
- Exportní delivery pack pro prezentaci a předání.
- Prezentační režim se zaměřením na jasnou strukturu a navigaci.

## Struktura webu
- Struktura stránek nebyla explicitně detekována.

## Shared assets
- none

## Hodnota projektu
- Vzniká konzistentní multi-page web, který jde bezpečně iterovat a opakovaně publikovat.
- Manifest + update mode snižují riziko chaotických změn mezi iteracemi.
